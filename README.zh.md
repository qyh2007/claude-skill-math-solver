# Claude Code Skill: 高等数学解题

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](./LICENSE)
[![Claude Code](https://img.shields.io/badge/Claude%20Code-skill-orange)](https://claude.ai/code)

面向 **Claude Code** 的高等数学解题技能，覆盖微积分与高等代数两大模块共 20 个专题。默认简洁输出，按需展开；支持图片转写、纠错、证明、概念讲解和 matplotlib 作图。

## 安装

```bash
git clone https://github.com/qyh2007/claude-skill-math-solver.git \
  ~/.claude/skills/math-solver
```

如果使用 ECC 生态：

```bash
cd ~/.claude/skills/ecc
git clone https://github.com/qyh2007/claude-skill-math-solver.git math-solver
```

安装后在 Claude Code 中通过 `/math-solver` 调用，或直接提数学问题——技能会自动匹配触发。

## 输出模式

| 模式 | 触发条件 | 输出内容 |
|------|----------|----------|
| **简洁模式**（默认） | 仅给出题目 | `解：` → 逐步推导 → `答：` |
| **详细模式** | "详细解释""为什么""检查我的过程"等 | 题型 → 思路 → 推导 → 易错点 → 答案 |
| **只给答案** | "只要答案""最终结果就行"等 | 仅输出答案，最多一句说明条件/单位/定义域 |

所有模式均遵循：
- 无废话（不说"我们来分析""综上所述"等过渡句）
- 公式用 `\(...\)` 或 `\[...\]` 内联书写，不在末尾单独罗列
- 引定理前验证前提条件（如洛必达需 0/0 或 ∞/∞）
- 多方法时默认给最常用考试方法，简要提替代思路
- 简单题自动省略"题型"和"易错点"
- 输出标签跟随用户提问语言（中文用 解/答/证明，英文用 Solution/Answer/Proof）
- 精确值优先：保留分数、根式、π、e、ln 2，仅题目要求时才用 ≈ 小数

## 核心行为

### 求解前检查 & 解后自检

解题前内部验证定义域和前置条件（分母、对数真数、根号、反三角函数范围、参数边界等），给出答案前内部自检方法条件、符号、边界和完整性，仅当影响结果或容易被忽略时才显式提醒。

### 纠错模式

用户上传自己的解题过程或问"错在哪""这样做对吗"时：

- 先检查已有步骤，不从头重做
- 判断结论是否正确；若错，定位第一个关键错误并解释原因
- 从错误处继续，不重复用户已做对的部分
- 若完全正确则简短确认，可选建议更规范的形式
- 格式：`判定：正确/错误/基本正确` → `问题：…` → `修正：…` → `答：…`

### 证明题

- 优先使用当前学习阶段的方法（早期章节知识优先于后期高级工具）
- 格式：`思路：…` → `证明：…` → `结论：…`
- 不写"显然""易知"等跳过关键推理的词

### 概念讲解

用户问概念含义、直观理解或易混概念的区别时：

- 先用具体例子和通俗语言解释，不直接堆形式化定义
- 辅助几何直观时用 matplotlib 画图展示
- 辨析易混概念时用反例说明（如"可微 vs C¹"：单变量等价，多变量不等价）

### 图片处理

1. 从图片提取题目文字和条件
2. 模糊不清处标注 `[不确定：此处似为 ...]`
3. 复述完整题目，请用户确认后再解题（防止 OCR 偏差）
4. 不描述图片排版、颜色或外观

## 示例

### 简洁模式

```
用户：求极限 lim(x→0) (sin x - x) / x³

解：
泰勒展开 sin x = x - x³/6 + x⁵/120 - ...
sin x - x = -x³/6 + O(x⁵)
原式 = lim (-x³/6 + O(x⁵)) / x³ = -1/6
答：-1/6
```

### 纠错模式

```
用户（上传解答过程）：∫xeˣdx = xeˣ + C

判定：错误
问题：分部积分只完成了 uv 项，漏掉了 -∫v du 部分；
      正确公式为 ∫u dv = uv - ∫v du
修正：
令 u = x, dv = eˣdx，则 du = dx, v = eˣ
∫xeˣdx = xeˣ - ∫eˣdx = xeˣ - eˣ + C = eˣ(x - 1) + C
答：eˣ(x - 1) + C
```

### 图片确认

```
题目转写：
设函数 f(x) = x³ - 3x + 1，求 f(x) 在区间 [-2, 2] 上的最大值和最小值。
请确认题目是否正确，确认后我开始解题。
```

用户确认"正确" → 然后解题：

```
解：
f'(x) = 3x² - 3 = 3(x-1)(x+1)，驻点 x = 1, x = -1
f(-2) = -1, f(-1) = 3, f(1) = -1, f(2) = 3
答：最大值 3，最小值 -1
```
